# PhDamage

A floating combat text replacement addon for World of Warcraft.

## Features

- Custom floating combat text display
- Configurable damage number formatting
- Support for different damage types and schools
- Smooth animations and positioning
- Layered architecture (Core / Engine / Data / Presentation)
- Multi-version support

## Supported Versions

- Retail
- MoP Classic
- TBC Anniversary
- Cataclysm Classic
- Classic Era
